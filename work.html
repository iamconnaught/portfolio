My Projects

Some of my projects

DADDIO Github

My Capstone project for General Assembly, DADDIO is a tool for men about to become fathers. It is part organizational tool, part informational resource. Users can add an upcoming baby, generate name suggestions and save those that they like, upload tasks for a to-do list as well as post and read about theirs or others' experiences during this time and during fatherhood.

ReactJs, JavaScript, NodeJS, Express, MongoDB, Mongoose, CSS, 3rd Party API

Music BucketList Github

Music Bucketlist comes from my love of live music, I wanted to be able to keep track of concerts I have attended, and keep a running list of bands I want to see in concert.

ReactJs, JavaScript, NodeJS, Express, MongoDB, Mongoose, CSS, 3rd Party API

My Chicago Github

My Chicago, which I worked on with TANIA I MEDINA PEREIRA is an app to discover and share experiences around the city of Chicago. Users can upload pictures, and locations to posts describing an experience.

JavaScript, NodeJS, Express, MongoDB, Mongoose, CSS

Escape Room Github

Escape Room is a game based off of my enjoyment of puzzles and escape rooms. The animation, which I created myself, was inspired by 80s/90s era video games like Pokemon and Legend of Zelda.

HTML, CSS, JavaScript, Canvas
